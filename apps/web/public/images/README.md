# Images Directory

Place all images in this folder. The following images are needed:

## Required Images

### Home Page Images

1. **Hero Image** (`hero-image.jpg` or `.png`)
   - Search: "African youth education empowerment" or "African students learning"
   - Should show diverse African youth, preferably in educational settings
   - Dimensions: 1920x1080 or similar wide format
   - Format: JPG or PNG

2. **About Us Image** (`about-us.jpg` or `.png`)
   - Search: "African community helping youth" or "African NGO volunteers"
   - Should show community support, mentorship, or helping hands
   - Dimensions: 1200x800 or similar
   - Format: JPG or PNG

3. **Feature Icons/Images** (Optional - can use icons instead)
   - `feature-youth.jpg` - Youth accessing opportunities
   - `feature-donor.jpg` - Donors/NGOs supporting youth
   - `feature-verification.jpg` - Verification process
   - `feature-impact.jpg` - Positive impact/transformation

4. **Stats/Impact Image** (`impact-image.jpg` or `.png`)
   - Search: "African youth success stories" or "graduation ceremony Africa"
   - Should show achievement, success, celebration
   - Dimensions: 1200x800
   - Format: JPG or PNG

5. **Testimonial/Story Images** (Optional)
   - `testimonial-1.jpg` - Success story image
   - `testimonial-2.jpg` - Another success story

### Dashboard Images

6. **Empty State Images** (Optional - can use illustrations)
   - `empty-opportunities.svg` or `.png` - For empty opportunities list
   - `empty-applications.svg` or `.png` - For empty applications list
   - `empty-documents.svg` or `.png` - For empty documents list

### General

7. **Logo** (`logo.png` or `.svg`)
   - RiseUp Africa logo (if you have one)
   - Dimensions: 200x200 or similar square format
   - Format: PNG with transparency or SVG

8. **Favicon** (`favicon.ico`)
   - Small icon for browser tab
   - Dimensions: 32x32 or 64x64
   - Format: ICO or PNG

## Image Guidelines

- **Format**: JPG for photos, PNG for images with transparency, SVG for logos/icons
- **Size**: Optimize images (compress) to keep file sizes reasonable (< 500KB each)
- **Dimensions**: Use high-resolution images (at least 2x the display size for retina)
- **Content**: Ensure images are respectful, diverse, and represent the target audience
- **Licensing**: Use images with appropriate licenses (Unsplash, Pexels, or your own)
